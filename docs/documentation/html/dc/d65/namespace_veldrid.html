class  CommandList
 A device resource which allows the recording of graphics commands, which can later be executed by a GraphicsDevice. Before graphics commands can be issued, the Begin method must be invoked. When the CommandList is ready to be executed, End must be invoked, and then GraphicsDevice.SubmitCommands(CommandList) should be used. NOTE: The use of CommandList is not thread-safe. Access to the CommandList must be externally synchronized. There are some limitations dictating proper usage and ordering of graphics commands. For example, a Framebuffer, Pipeline, vertex buffer, and index buffer must all be bound before a call to DrawIndexed(uint, uint, uint, int, uint) will succeed. These limitations are described in each function, where applicable. CommandList instances cannot be executed multiple times per-recording. When executed by a GraphicsDevice, they must be reset and commands must be issued again. See CommandListDescription. More...

◆ BlendFactor

enum Veldrid.BlendFactor : byte
strong

Controls the influence of components in a blend operation.

Enumerator
Zero 

Each component is multiplied by 0.

One 

Each component is multiplied by 1.

SourceAlpha 

Each component is multiplied by the source alpha component.

InverseSourceAlpha 

Each component is multiplied by (1 - source alpha).

DestinationAlpha 

Each component is multiplied by the destination alpha component.

InverseDestinationAlpha 

Each component is multiplied by (1 - destination alpha).

SourceColor 

Each component is multiplied by the matching component of the source color.

InverseSourceColor 

Each component is multiplied by (1 - the matching component of the source color).

DestinationColor 

Each component is multiplied by the matching component of the destination color.

InverseDestinationColor 

Each component is multiplied by (1 - the matching component of the destination color).

BlendFactor 

Each component is multiplied by the matching component in constant factor specified in BlendStateDescription.BlendFactor.

InverseBlendFactor 

Each component is multiplied by (1 - the matching component in constant factor specified in BlendStateDescription.BlendFactor).
