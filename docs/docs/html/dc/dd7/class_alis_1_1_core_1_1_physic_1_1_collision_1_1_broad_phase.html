◆ QuerySegment()

unsafe int Alis.Core.Physic.Collision.BroadPhase.QuerySegment ( Segment  segment,
object[]  userData,
int  maxCount,
SortKeyFunc  sortKey 
)

Query a segment for overlapping proxies, returns the user data and the count, up to the supplied maximum count. If sortKey is provided, then it is a function mapping from proxy user Data to distances along the segment (between 0 ans 1) Then the returned proxies are sorted on that, before being truncated to maxCount The sortKey of a proxy is assumed to be larger than the closest point inside the proxy along the segment, this allows for early exits Proxies with a negative sortKey are discarded
