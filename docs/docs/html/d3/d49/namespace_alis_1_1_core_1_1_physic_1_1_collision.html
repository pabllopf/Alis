class  ManifoldPoint
 A manifold point is a contact point belonging to a contact manifold. It holds details related to the geometry and dynamics of the contact points. The local point usage depends on the manifold type: -Circles: the local center of circleB -FaceA: the local center of cirlceB or the clip point of polygonB -FaceB: the clip point of polygonA This structure is stored across time steps, so we keep it small.
