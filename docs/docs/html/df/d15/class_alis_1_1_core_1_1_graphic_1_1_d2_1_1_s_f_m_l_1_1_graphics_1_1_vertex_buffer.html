◆ Update() [1/4]

bool Alis.Core.Graphic.D2.SFML.Graphics.VertexBuffer.Update ( Vertex[]  vertices,
uint  vertexCount,
uint  offset 
)

Update a part of the buffer from an array of vertices offset is specified as the number of vertices to skip from the beginning of the buffer. If offset is 0 and vertexCount is equal to the size of the currently created buffer, its whole contents are replaced. If offset is 0 and vertexCount is greater than the size of the currently created buffer, a new buffer is created containing the vertex data. If offset is 0 and vertexCount is less than the size of the currently created buffer, only the corresponding region is updated. If offset is not 0 and offset + vertexCount is greater than the size of the currently created buffer, the update fails. No additional check is performed on the size of the vertex array, passing invalid arguments will lead to undefined behavior.

Parameters
verticesArray of vertices to copy to the buffer
vertexCountNumber of vertices to copy
offsetOffset in the buffer to copy to
