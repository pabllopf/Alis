# Pattern: RS2007 - Analyzer Release File Format

## Rule

external_roslyn:RS2007

## Description

Analyzer release file has a missing or invalid release header.

## Fix

1. Ensure `## Unshipped` or `## Shipped` header exists as first line
2. Add proper markdown table with 5 columns: Rule ID, Category, Severity, Notes
3. Create both Shipped and Unshipped files in all analyzer projects
4. Ensure projects have `<EnforceExtendedAnalyzerRules>true</EnforceExtendedAnalyzerRules>`

## Example

```markdown
## Unshipped

| Rule ID | Category | Severity | Notes |
|---------|----------|----------|-------|
| ALIS001 | AOT/Reflection | Error | AotReflectionAnalyzer |
```
