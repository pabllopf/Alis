# Alis.App.Hub

## Overview
Game hub/launcher application for managing and launching ALIS games.

## Project Details
- **Layer**: 1_Presentation
- **Type**: Application (Desktop)
- **Framework**: net8.0
- **Purpose**: Game launcher and library manager

## Dependencies
- Alis.App.Core (2_Application)
- Alis.Extension.Network (for online features)
- Alis.Extension.Updater (for game updates)

## Key Files
- `Program.cs` — Hub entry point
- `MainWindow.xaml` — Hub UI
- `GameLibrary.cs` — Game library management

## Notes
- Manages game installations
- Handles game updates
- Provides game launch interface
