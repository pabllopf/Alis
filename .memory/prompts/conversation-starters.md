## For New AI Sessions
When starting work on ALIS, consider these context questions:

1. "What architectural layer does this feature belong to?"
2. "Which projects reference {ProjectName}?"
3. "What does the source generator for {Aspect} produce?"
4. "How do I add a new extension to ALIS?"
5. "What's the dependency flow between layers?"

## For Human Developers
Onboarding questions:

1. "How does the source generator cascade work?"
2. "What's the difference between 3_Structuration and 4_Operation?"
3. "How do I run a specific game sample?"
4. "Where should I add new engine functionality?"
5. "How are NuGet packages structured with native libraries?"
