# Alis

C# game engine/framework organized into 6 architectural layers with aspect-oriented design.

## Summary
- **Language**: C# (.NET)
- **Architecture**: 6-layer modular design
- **Solution files**: 8 (.slnx) for different build targets
- **Sample games**: 12 (each with Web + Desktop platforms)
- **Extensions**: 18+ modular extensions
