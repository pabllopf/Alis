# Multi-Platform Samples

Alis includes 12 sample games, each targeting Web and Desktop platforms:

## Sample Games

| Sample | Description |
|--------|-------------|
| King.Platform | Platformer game |
| Flappy.Bird | Flappy Bird clone |
| Space.Simulator | Space simulation |
| Dino | Dino runner |
| Empty | Minimal template |
| Pong | Pong clone |
| SplitCamera | Camera system demo |
| Asteroid | Asteroids clone (also iOS/Android) |
| Rogue | Roguelike game |
| Snake | Snake clone |
| RuinsOfTartarus | RPG demo |
| Egg | Egg game |
| Inefable | Abstract game |

## Platform Variants
Each sample has `web/` and `desktop/` sub-projects. Asteroid additionally targets iOS and Android.
