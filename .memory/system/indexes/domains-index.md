# Domains Index

## Bounded Contexts

| Domain | Projects | Description |
|---|---|---|
| Core Engine | Alis.Core, Alis.Core.Ecs, Alis.Core.Graphic | Fundamental engine capabilities |
| Game Systems | Alis.Core.Audio, Alis.Core.Physic | Audio and physics simulation |
| AOP Framework | Alis.Core.Aspect.* | Aspect-oriented programming utilities |
| Extensions | Alis.Extension.* | Platform-specific extensions |
| Applications | Alis.App.* | User-facing applications |
| Samples | Alis.Sample.* | Example implementations |

## Domain Responsibilities

1. **Core Engine**: Base abstractions, entity management, rendering
2. **Game Systems**: Audio playback, physics simulation
3. **AOP Framework**: Cross-cutting concerns (logging, memory, data)
4. **Extensions**: Platform bindings, third-party integrations
