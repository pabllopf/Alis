AOT-safe source generators for compile-time code generation.

## Generator Projects

| Generator | Location | Produces |
|-----------|----------|----------|
| **Ecs.Generator** | `4_Operation/Ecs/generator/` | ECS component/system registry |
| **Graphic.Generator** | `4_Operation/Graphic/generator/` | Graphics primitive registry |
| **Memory.Generator** | `6_Ideation/Memory/generator/` | Memory resource accessors |
| **Data.Generator** | `6_Ideation/Data/generator/` | Data structure builders |
| **Fluent.Generator** | `6_Ideation/Fluent/generator/` | Fluent API builders |

## Generator Architecture

### Base Generator Pattern
```csharp
[Generator]
public class AlisSourceGenerator : ISourceGenerator
{
    public void Initialize(GeneratorInitializationContext context)
    {
        // Register attributes
    }
    
    public void Execute(GeneratorExecutionContext context)
    {
        // Generate code
    }
}
```

### Generated Code Examples

#### Component Registry (ECS)
```csharp
// Generated by ComponentUpdateTypeRegistryGenerator
public static partial class AlisComponentRegistry
{
    public static void RegisterComponents(UpdateRegistry registry)
    {
        registry.Register<Position>();
        registry.Register<Velocity>();
    }
}
```

#### Resource Accessors (Memory)
```csharp
// Generated by ResourceAccessorGenerator
public static partial class AssemblyLoader
{
    public static IEnumerable<Type> GetResourceTypes()
    {
        return typeof(ResourceMarker).Assembly.GetTypes()
            .Where(t => t.IsDefined(typeof(ResourceAttribute)));
    }
}
```

## AOT Compatibility

All generators produce:
- No runtime IL emit
- No reflection-based code generation
- Static, compile-time generated code
- Full Native AOT support
