## Generated Code Characteristics

- **AOT-safe** - No runtime IL emit, Native AOT compatible
- **Static** - All code generated at compile time
- **Type-safe** - Compile-time error detection
- **Diagnostics** - Clear error messages (ALIS0xxx IDs)
