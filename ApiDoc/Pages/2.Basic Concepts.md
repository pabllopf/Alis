# 2. Basic Concepts

Antes de poder empezar a desarrollar un videojuego en Alis, se debe de
tener claros los siguientes aspectos.

### Scene

Un videojuego está compuesto de uno o múltiples escenarios denominados
'Scenes'. Una escena o 'Scene', está compuesto por uno o múltiples
objetos denominados 'GameObjects'.

### GameObject

Un GameObject es un elemento del videojuego que se encuentra dentro de
un escenario y que contiene componentes que le atribuyen características
especiales que lo definen y diferencian del resto.

### Transform

El transform es una propiedad especial de los gameobjects que representa
su posición, rotación y escala de tamaño.

### Component

Un componente o 'Component' representa una característica especial de un
gameobject. En Alis existen 5 componentes implementados que permiten
agilizar el desarrollo. Aunque también es posible crear tus propios
componentes.

1.  Sprite: Es un componente que permite asignar una imagen a un
    gameobject.

2.  Camera: Este componente permite asignar una cámara con determinadas
    características a un gameobject.

3.  Audiosource: Este componente permite asignar sonidos a los
    GameObjects.

4.  Animator: Este componente permite crear arboles de animaciones que
    podrían ser utilizados por otros componentes para integrar
    movimiento a los GameObjects.

5.  Collision: Este componente permite generar colisiones y hacer que
    los GameObjects puedan interaccionar entre sí.
