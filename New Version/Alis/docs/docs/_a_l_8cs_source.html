347  /* Source
348  * Sources represent individual sound objects in 3D-space.
349  * Sources take the PCM buffer provided in the specified Buffer,
350  * apply Source-specific modifications, and then
351  * submit them to be mixed according to spatial arrangement etc.
352  *
353  * Properties include: -
354  *
355 
356  * Position AL_POSITION ALfloat[3]
357  * Velocity AL_VELOCITY ALfloat[3]
358  * Direction AL_DIRECTION ALfloat[3]
359 
360  * Head Relative Mode AL_SOURCE_RELATIVE ALint (AL_TRUE or AL_FALSE)
361  * Looping AL_LOOPING ALint (AL_TRUE or AL_FALSE)
362  *
363  * Reference Distance AL_REFERENCE_DISTANCE ALfloat
364  * Max Distance AL_MAX_DISTANCE ALfloat
365  * RollOff Factor AL_ROLLOFF_FACTOR ALfloat
366  * Pitch AL_PITCH ALfloat
367  * Gain AL_GAIN ALfloat
368  * Min Gain AL_MIN_GAIN ALfloat
369  * Max Gain AL_MAX_GAIN ALfloat
370  * Inner Angle AL_CONE_INNER_ANGLE ALint or ALfloat
371  * Outer Angle AL_CONE_OUTER_ANGLE ALint or ALfloat
372  * Cone Outer Gain AL_CONE_OUTER_GAIN ALint or ALfloat
373  *
374  * MS Offset AL_MSEC_OFFSET ALint or ALfloat
375  * Byte Offset AL_BYTE_OFFSET ALint or ALfloat
376  * Sample Offset AL_SAMPLE_OFFSET ALint or ALfloat
377  * Attached Buffer AL_BUFFER ALint
378  *
379  * State (Query only) AL_SOURCE_STATE ALint
380  * Buffers Queued (Query only) AL_BUFFERS_QUEUED ALint
381  * Buffers Processed (Query only) AL_BUFFERS_PROCESSED ALint
382  */
