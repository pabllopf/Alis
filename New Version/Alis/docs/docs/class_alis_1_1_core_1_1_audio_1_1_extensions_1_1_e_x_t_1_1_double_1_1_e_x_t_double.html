DoubleBufferFormat  format,

This function fills a buffer with audio buffer. All the pre-defined formats are PCM buffer, but this function may be used by extensions to load other buffer types as well.

Parameters
bidbuffer Handle/Name to be filled with buffer.
formatFormat type from among the following: DoubleBufferFormat.Mono, DoubleBufferFormat.Stereo.
bufferPointer to a pinned audio buffer.
bytesThe size of the audio buffer in bytes.
freqThe frequency of the audio buffer.
